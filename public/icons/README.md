# App Icons

To complete the PWA setup, please add app icons in the following sizes:
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

You can use a tool like https://realfavicongenerator.net/ to generate these from your logo.

For now, the PWA will work without icons, but the install prompt will look better with them.
